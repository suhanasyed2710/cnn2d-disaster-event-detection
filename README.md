# CNN2D-Based Deep Learning Framework with Blockchain-Enabled Trust Verification for Real-Time Disaster Event Detection

## Overview
This project detects disaster-related events from social media using CNN2D Deep Learning and verifies trust using Blockchain technology.

## Technologies Used
- Python
- CNN2D
- NLP
- Blockchain
- Smart Contracts
- Machine Learning

## Features
- Disaster event detection
- Fake information identification
- Blockchain verification
- Real-time monitoring
- Secure event validation

## Modules
- Data Collection
- Data Preprocessing
- CNN2D Model
- Blockchain Validation
- Event Classification
